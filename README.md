# Ben Abdesselem Youssef - Portfolio

A premium, bilingual (English/French) portfolio website built with **Next.js 16**, **React 19**, and **Tailwind CSS 4**. This project showcases professional experience, technical skills, and educational background with a focus on modern aesthetics and smooth user interactions.

## 🚀 Features

- **🌍 Bilingual Support**: Seamlessly switch between English and French.
- **🎨 Premium Design**: Modern, clean, and responsive UI with dark mode support.
- **⚡ Performance**: Built with Next.js App Router for optimal speed and SEO.
- **📱 Responsive**: Fully optimized for mobile, tablet, and desktop devices.
- **✨ Micro-animations**: Subtle transitions and hover effects for an enhanced user experience.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **UI Components**: Custom-built with React 19.
- **Icons**: [Lucide React](https://lucide.dev/) (implied or used in components)

## 📁 Project Structure

- `/app`: Next.js App Router pages and layouts.
- `/app/components`: Reusable UI components (Navbar, Hero, Experience, etc.).
- `/app/context`: Language and theme state management.
- `/app/data`: Professional data and translation strings.
- `/public`: Static assets including images and logos.

## 🏁 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm, yarn, pnpm, or bun

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [repository-url]
    cd portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open in browser:**
    Visit [http://localhost:3000](http://localhost:3000) to see the result.
